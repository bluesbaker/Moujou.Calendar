# Moujou.Calendar
The Calendar view for the UI Xamarin.forms
### Installation
```Clone repository and have fun```

### Usage
Add in header
```xml
<...
xmlns:moujou="clr-namespace:Moujou.Calendar;assembly=Moujou.Calendar"
...>
```
...and...
```xml
...
<moujou:CalendarView Year="2021" Month="1" Day="31"/>
...
```
